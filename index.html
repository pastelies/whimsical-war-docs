ウィムジカルウォー 復活プロジェクト
終了したモバイルゲームを身内向けに再現するプロジェクト。タワーディフェンス＋陣地塗りのリアルタイムバトルゲーム。
Unity 6.3 LTS Node.js + SQLite WebSocket 通信 Oracle Cloud（予定）
Android（先行開発中）
iOS（後続で対応予定）
Unity を使用しているため Android / iOS の両対応が可能。まず Android で完成させ、動作確認後に iOS 版をビルドする予定です。iPhone ユーザも全員遊べるようにすることがゴールです。

確定している技術仕様（主要ポイント）
フィールド
28×12タイル、橋3本（各2×2）
カード構成
8枚デッキ / 手札4枚 + 待機1枚
エリクサー
3秒で1回復、残り1分で2倍
サブタワー破壊
1本目+50pt、2本目+150pt（計200pt）
通信方式
WebSocket、200ms間隔でstate配信
カードデータ
ログイン時にサーバーから取得（バランス調整可能）
